# Rock_Paper_Scissor_Game
- Rock Paper Scissors | Classic Arcade Game
- A fully interactive Rock Paper Scissors game with animated battles, score tracking, and match history.
- Live Demo: [ https://redluzon.github.io/Rock_Paper_Scissor_Game/ROCK_PAPER_SCISSOR/HTML-FILE/start.html]

Features
- Animated hand gestures with shake and win/lose effects
- Real-time score tracking (Player vs Computer)
- Battle history log showing each round's outcome
- Responsive design for mobile, tablet, and desktop
- Smooth transitions and floating background shapes
- Reset game functionality

Tech Stack
- HTML5, CSS3, JavaScript
- CSS Grid & Flexbox for layouts
- CSS keyframe animations
- Local state management

Game Rules
- ✊ Rock beats Scissors
- ✋ Paper beats Rock
- ✌️ Scissors beats Paper

Pages Included
- Start Screen with game preview and CTA button
- Main Game Arena with interactive controls
